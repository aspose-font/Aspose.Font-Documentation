## Major Features

Aspose.Font for  C++ 21.8 is based on the latest .Net version 21.8 with added converters CFF to TTF/WOFF and supporting all tables for TTF and WOFF/WOFF2.

Glyphs and TtfFontDefinitionParser optimized.
